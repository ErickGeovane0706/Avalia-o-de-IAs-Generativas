# Roulette Canvas Game

## Overview
This project is a simulation of a roulette game implemented using HTML5 Canvas and Vanilla JavaScript. It features realistic physics, including friction and gravity, to create an engaging user experience. The game is designed with a premium aesthetic, utilizing dark colors and gold accents.

## Project Structure
```
roulette-canvas
├── src
│   ├── index.html       # HTML structure for the roulette game
│   ├── style.css        # Styles for the roulette game
│   └── script.js        # Game logic implemented in JavaScript
├── package.json         # Configuration file for npm
└── README.md            # Documentation for the project
```

## Features
- Realistic physics simulation with friction and gravity.
- Object-Oriented Programming (OOP) design with separate classes for `Roulette` and `Ball`.
- Premium visual aesthetics with dark colors and gold accents.
- Circular collision detection for the ball and roulette pins.

## Getting Started
To run the roulette game, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd roulette-canvas
   ```

3. Open the `src/index.html` file in a web browser to start the game.

## Dependencies
This project does not require any external libraries and is built using Vanilla JavaScript.
